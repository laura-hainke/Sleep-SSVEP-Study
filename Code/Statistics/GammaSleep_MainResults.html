Statistical test

Additionally, in the experimental condition, 40 Hz power was expected to differ between stages (H5a). A repeated measures ANOVA, Greenhouse-Geisser-corrected for sphericity deviations, confirmed this hypothesis (p = 2.56e-169). The effect size was large, with an eta squared of 0.9 (CI = [0.87 ; 1]). However, post-hoc tests reveal that only stage W differs from all other stages, with no significant differences between the sleep stages N2, N3, and REM (pW-N2 = 1.37e-24, pW-N3 = 2.71e-26, pW-REM = 2.01e-23, all other p = 1). We can conclude that 40 Hz power during stimulation is largest in W, and similarly large between sleep stages.

Statistical test

Regarding subjective sleep quality quantified as the GSQS sum score, we postulated that it would differ between the control and experimental condition. An undirected paired t-test confirmed this hypothesis (p = 0.01), indicating that sleep quality was perceived to be higher in the experimental (Meanexp = 2.73, SDexp = 2.56) than in the control (Meancon = 4.57, SDcon = 3.28) night. The effect size was small, with a Cohens dz of 0.48 (CI = [0.11 ; 0.85]).

Lastly, Time per Stage descriptively points to a sleep stage distribution common in healthy, young sleepers. Additionally, in the experimental condition, SSVEP amplitude was expected to differ between stages (H5b). A Friedman ANOVA confirmed this hypothesis (p = 4.04e-13). A Kendalls w of 0.68 (CI = [0.62 ; 1]) indicates substantial agreement. However, post-hoc tests reveal that only stage W differs from all other stages, with no significant difference between the sleep stages N2, N3, and REM (pW-N2 = 1.5e-12, pW-N3 = 5.15e-14, pW-REM = 1.54e-10, all other p = 1). We can conclude that SSVEP amplitude during stimulation is largest in W, and similarly large between sleep stages.
